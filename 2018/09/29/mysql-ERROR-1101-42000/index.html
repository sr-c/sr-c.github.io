mysql ERROR 1101 (42000)

发表于 2018-09-29

新建数据表时出现不允许缺省值的报错

问题描述

新建数据表时出现报错

这是由于mysql处于strict mode, 不允许text或blod字段有缺省值。因此，我们需要修改mysql为非严格模式。

解决方案

一般建议修改mysql的配置文件my.ini,将其中sql-mode这一行注释掉即可。
但在LinuxMint中，mysql的默认配置文件中没找到这一行。因此，手动在mysql中修改sql_mode
