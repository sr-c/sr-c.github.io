【debug】C stack usage is too close to the limit

发表于 2019-02-16

运行Rscript读取数据时提示报错

问题描述

这似乎是由于数据过大导致

解决方案

这个值本应该等于10240*1024=10485760，但我们得到的9961472比这略小一些，在此不明原因。

解决方案是手动设定C stack

然后重新运行程序，即可解决
