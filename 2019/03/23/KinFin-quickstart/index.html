使用KinFin进行比较基因组分析

发表于 2019-03-23

使用KinFin整合下游的比较基因组分析

安装

安装程序会下载一些数据，并使用pip安装相应的模块。但若是没有管理员权限，还是使用conda来配置环境比较方便。

软件的环境要求python模块如下

scipy==0.19.0
matplotlib==2.0.2
docopt==0.6.2
networkx==1.11
powerlaw==1.4.1
ete3==3.0.0b35

出现帮助信息表明软件安装完成

此外，还可以安装一些其他软件

测试

KinFin的软件路径中提供了测试数据./test，但我们还可以使用教程中的数据来练习。
